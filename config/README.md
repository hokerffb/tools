# 配置文件

## Linux: Ubuntu 18.04/WSL

|配置文件|路径|
:-|:-|
|vimrc.local|/etc/|

## MacOS 10.15

|配置文件|路径|
:-|:-|
|.zshrc|~/|
|vimrc.local|/etc/|
